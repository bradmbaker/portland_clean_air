DEQ Permits

The buisness located at this address is listed as AjinomotoWindsor, Inc. and the North American Industry Classification Code System code for the business according to the DEQ is 221330.

This is a DEQ General Type 34 permit, DEQ Basic Permit

Basic emitters are the least significant air pollution emitters tracked by Oregon DEQ.

All permit types can be found on The Oregon DEQ Website.

Files with aqb and three digits are the base permit application. A file with aqrb and three digits is the annual reporting form.

Basic ACDP permits are governed by OAR 340 Division 200 to 268 and no specific permits are issued.
